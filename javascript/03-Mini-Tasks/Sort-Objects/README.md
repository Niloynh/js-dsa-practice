## Employee Management Data Utilities (Real-World JS Project)

This module includes real-world utility functions used in admin dashboards, HR systems, and company portals.

### Features
- Sort employees by salary (desc)
- Filter frontend developers
- Search employees by name (case insensitive)
- Sort employees by experience
- Combined filter + sort operations

### Example Use Cases
- HR dashboards  
- Salary management  
- Experience ranking  
- Search bar implementation  

This project helps strengthen JavaScript skills related to sorting, filtering, and real-world data handling.
